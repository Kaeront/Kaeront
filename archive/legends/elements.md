# Элементы
Магия — это не просто абстрактная энергия, а физический закон, на котором строится взаимодействие всего живого. Согласно историческим записям, до падения Регнусолиса великие Легенды прошлого использовали эти силы для поддержания баланса, однако сейчас, когда наёмники Баддой используют тёмные Разломы для искажения реальности, понимание элементальных реакций стало жизненно важным навыком для выживания каждого.
## Характеристики элементов
 * <span style="color: #f55; font-family: 'Minecraft', sans-serif">🔥 Пиро</span>: Элемент первородного жара. Используется для дестабилизации биологических характеристик противника.
 * <span style="color: #55f; font-family: 'Minecraft', sans-serif">💧 Аква</span>: Гибкая стихия, позволяющая проводить электричество или создавать основы для сложных реакций.
 * <span style="color: #0aa; font-family: 'Minecraft', sans-serif">🌪️ Вентус</span>: Катализатор. Его главная задача в бою — расширять радиус действия других статусов через Рассеивание.
 * <span style="color: #a0a; font-family: 'Minecraft', sans-serif">⚡ Фулгор</span>: Энергия высокого напряжения, идеальна против бронированных или синтетических врагов.
 * <span style="color: #5f5; font-family: 'Minecraft', sans-serif">🌿 Флора</span>: Основа для создания полевых бутонов, которые могут резонировать с вторичными элементами.
 * <span style="color: #5ff; font-family: 'Minecraft', sans-serif">❄️ Глациес</span>: Элемент контроля. Основная функция — замедление (заморозка) или снижение защиты врага.
 * <span style="color: #fa0; font-family: 'Minecraft', sans-serif">🪨 Терра</span>: Стабилизирующая сила. Используется для создания защитных щитов (кристаллизация) или дебаффов, привязанных к поверхности земли.
