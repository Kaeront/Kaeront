# Политика конфиденциальности и обработки персональных данных проекта «Kaeront»
###### Последнее изменение: 13.03.2026
### 1. Общие положения
1.1. Настоящий документ (далее — «Политика») определяет порядок обработки и защиты информации о физических лицах (далее — «Пользователи»), использующих ресурсы и серверную инфраструктуру проекта «Kaeront» (далее — «Проект») в лице Администратора Амира Кашапова.
1.2. Проект обеспечивает соблюдение прав и свобод человека при обработке его персональных данных, включая защиту прав на неприкосновенность частной жизни и личную тайну.
### 2. Состав и сбор информации
2.1. В рамках обеспечения функционирования технической инфраструктуры Проекта могут обрабатываться следующие данные:
  * **Технические идентификаторы:** Сетевой адрес (IP), уникальный идентификатор пользователя (UUID), системные логи подключений.
  * **Профильные данные:** Псевдоним пользователя (Никнейм), идентификатор в мессенджере Telegram (при добровольном обращении пользователя в службу поддержки).
  * **Логи активности:** Данные о внутрисистемных действиях, необходимые для обеспечения безопасности и стабильности серверной инфраструктуры.
2.2. Веб-ресурс Проекта функционирует без использования технологий Cookie и систем скрытого профилирования поведения пользователей.
### 3. Цели обработки персональных данных
3.1. Обработка данных осуществляется на законных основаниях для достижения следующих целей:
  * Предоставление доступа к функционалу серверной инфраструктуры Проекта;
  * Оказание технической поддержки и оперативное рассмотрение обращений Пользователей;
  * Мониторинг безопасности, предотвращение несанкционированного доступа и деструктивных действий в отношении системы.
### 4. Обработка финансовых данных
4.1. Любые добровольные взносы (пожертвования) на поддержку Проекта осуществляются через сторонний независимый платежный сервис **CloudTips**.
4.2. Администрация Проекта не имеет доступа к банковским реквизитам, данным платежных карт и иным финансовым сведениям Пользователей. Процесс обработки указанных данных регулируется исключительно политикой конфиденциальности платежного сервиса.
### 5. Меры по обеспечению безопасности
5.1. Для защиты информации Пользователей применяются современные технические и организационные меры, включая использование защищенных протоколов передачи данных (SSL/TLS) и строгий контроль доступа к серверным журналам.
5.2. Персональные данные уничтожаются по достижении целей их обработки или в случае получения законного требования Пользователя об их удалении.
### 6. Заключительные положения
6.1. Проект «Kaeront» является независимой технической площадкой. Все упоминания стороннего ПО (в т.ч. Minecraft) приведены в ознакомительных целях. Проект не связан с Mojang AB или Microsoft.
6.2. По вопросам уточнения или удаления персональных данных Пользователь может обратиться по адресу: **admin@kaeront.ru** или через Telegram: **@KaerontMC**.
